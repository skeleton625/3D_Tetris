해당 소스 코드를 실행하기 위한 환경 설정에 문제가 생겨 소스 파일만 올림.

* 환경 설정
프로젝트 / 디버그
	- 해당 프로젝트 속성
	- 구성 관리자
		- 새로 만들기(아무 이름이나)
	- C/C++
		- 일반
			- 추가 포함 디렉터리
				- glutdlls37beta 폴더 주소 입력
	- 링커
		- 일반
			- 추가 라이브러리 디렉터리
				- glutdlls37btea 폴더 주소 입력
		- 입력
			- 추가 종속성
				- "편집"을 눌러 맨 위 입력창에 "glut32.lib" 입력

* SAFESEH 오류 발생 경우
프로젝트 / 디버그
	- 링커
		- 고급
			- 이미지에 안전한 예외 처리기 포함
				- "아니오"로 설정

* 사용법
[특수 키]
'z' KEY - x축 회전
'X' KEY - y축 회전
'C' KEY - z축 회전
'A' KEY - 아래로 이동
'S' KEY - 전체 회전

[BLOCK 이동]
키보드 방향키

[GAME OVER된 경우]
ESC KEY - 게임 재시작
